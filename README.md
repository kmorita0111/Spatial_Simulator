# Spatial SBML Simulator with HDF5+OpenCV
## How to Compile
### Dependency
- [libSBML 5.11.4](http://sbml.org/Software/libSBML "libSBML")
- [HDF5](https://www.hdfgroup.org/HDF5/ "hdf5")
- [OpenCV]("http://opencv.org/" "opencv") 3.0

インストールはmacportsとかで各自頑張って
